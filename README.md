# University CLI Management System

Консольний застосунок для управління університетом.
Програма дозволяє працювати зі студентами, викладачами, курсами та зарахуваннями на курси.

## Опис проєкту

Цей проєкт реалізує просту систему управління університетом через консольний інтерфейс.
Користувач може додавати, переглядати, оновлювати та видаляти дані, а також виконувати пошук, фільтрацію та формувати звіти.

Усі дані зберігаються в пам’яті програми під час її виконання.

## Основний функціонал
1. Студенти
    - Додавання нового студента
    - Перегляд усіх студентів
    - Оновлення даних студента
    - Видалення студента
    - Зміна статусу студента
    - Фільтрація студентів за статусом
    - Фільтрація студентів за роком навчання
    - Сортування студентів за ПІБ за допомогою bubble sort
    - Пошук студента за частиною ПІБ або email


2. Викладачі
    - Додавання викладача
    - Перегляд усіх викладачів
    - Оновлення даних викладача
    - Видалення викладача
    - Використання enum TeacherPosition


3. Курси
    - Додавання курсу
    - Перегляд усіх курсів
    - Оновлення даних курсу
    - Видалення курсу
    - Фільтрація курсів за викладачем
    - Фільтрація курсів за кількістю кредитів


4. Зарахування
    - Створення зарахування студента на курс
    - Пошук зарахування за ID
    - Видалення зарахування
    - Виставлення оцінки за 100-бальною шкалою
    - Автоматичне перетворення балів у enum Grade
    - Позначення оплати


5. Звіти та пошук
    - Пошук студента за ПІБ або email
    - Перегляд зарахувань конкретного студента
    - Виведення транскрипту студента
    - Показ неоплачених зарахувань
    - Розрахунок середнього GPA по курсу і семестру
    - Топ-N студентів за GPA
